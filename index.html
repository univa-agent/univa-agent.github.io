„ÄåMulti-Agent Architecture„ÄçPlan & Act Agents

UniVA employs a sophisticated dual-agent architecture where the Plan Agent strategizes and the Act Agent executes,
enabling complex video workflows that adapt based on intermediate results.

Plan Agent - DeepSeek-R1-0528

Receives user input and generates detailed, complete plans. Sends execution steps to Act Agent and dynamically updates plans based on execution results.

Act Agent - Claude-Sonnet-4

Receives commands from Plan Agent, selects appropriate tools, executes operations, and returns results for plan refinement.

System Architecture

UniVA revolutionizes video creation by introducing an intelligent multi-agent system that understands the iterative nature of creative workflows. Unlike traditional single-task models that excel in isolation, UniVA orchestrates multiple state-of-the-art models through a sophisticated planning and execution framework. The Plan Agent, powered by DeepSeek-R1, interprets user intent and creates comprehensive strategies, while the Act Agent, powered by Claude-Sonnet-4, precisely executes tasks using an extensive toolkit. This architecture enables seamless transitions between different video operations - from initial generation to editing, style transfer, and final composition. The system's key innovation lies in its content-aware workflow management, where each step's output directly influences subsequent operations, ensuring coherent and professional results.
